# 09 - RGB LED Example

This is a small example of a three pin RBG LED.

## Pins

01 - Resistor > Power

11 - Resistor > Red leg

12 - Resistor > Blue leg

13 - Resistor > Green leg

## Running

1. npm install in the folder
2. COLOUR={R|G|B} node index

This defaults to red - so node index can be ran.
